# Next.js App Router

## Instructions

   - Separate routes by feature

   - Group routes with `(auth)` or `(dashboard)`
   - Use `not-found.tsx` and `error.tsx`

## Best Practices
- Avoid heavy logic in client components
- Fetch data in server components
- Keep layouts thin and reusable
